[name]

This class represents a basic control model which is typically known from DCC tools like Blender. [name] is used in the official [link:https://threejs.org/editor/ three.js editor]. It is a simplified version of [page:OrbitControls].

Constructor

[name]( [param:Camera object], [param:HTMLDOMElement domElement] )

[page:Camera object]: The camera to be controlled.

[page:HTMLDOMElement domElement]: (optional) The HTML element used for event listeners. By default this is the whole document, however if you only want the controls to work over a specific element (e.g. the canvas) you can specify that here.

Creates a new instance of [name].

Events

change

Fires when the camera has been transformed by the controls.

Properties

[property:Vector3 center]

The focus or target point of the controls.

[property:HTMLDOMElement domElement]

The HTMLDOMElement used to listen for mouse / touch events. This must be passed in the constructor; changing it here will not set up new event listeners. Default is the whole document.

[property:Boolean enabled]

Whether or not the controls are enabled.

[property:Camera object]

The camera to be controlled.

[property:Number panSpeed]

The panning speed. Default is *0.002*.

[property:Number rotationSpeed]

The rotation speed. Default is *0.005*.

[property:Number zoom]

The zoom speed. Default is *0.1*.

Methods

[method:null focus] ( [param:Object3D Object3D] )

[page:Object3D Object3D]: The 3D object that should be focused on.

Focuses the center point of the given 3d object.

[method:null pan] ( [param:Vector3 delta] )

[page:Vector3 delta]: The amount of panning that should be applied.

Performs panning according to the given delta value and the respective panning speed.

[method:null rotate] ( [param:Vector3 delta] )

[page:Vector3 delta]: The amount of orbiting that should be applied.

Performs orbiting according to the given delta value and the respective rotation speed. When orbiting, the camera always looks at the [page:.center center] vector.

[method:null zoom] ( [param:Vector3 delta] )

[page:Vector3 delta]: The amount of zooming that should be applied.

Performs zooming (in fact dollying) according to the given delta value and the respective zoom speed.

[method:null dispose] ()

Should be called if the controls is no longer required.
